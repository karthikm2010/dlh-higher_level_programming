# Python Control Flow, Loops & Functions

This repository contains my exercises on **Python control flow, loops, and functions** as part of the AI Academy course at DLH.

---

## 📘 Objective

To practice Python fundamentals including:

* Conditional statements
* Loops
* String formatting
* Character handling with ASCII values
* Basic functions
* Simple algorithmic problems

---

## 🧩 Tasks Covered

### 🔀 Conditions

* `0-positive_or_negative.py` – Check if a number is positive, zero, or negative
* `1-last_digit.py` – Print and evaluate the last digit of a number

---

### 🔁 Loops & Formatting

* `2-print_alphabet.py` – Print lowercase alphabet
* `3-print_alphabt.py` – Print alphabet excluding `q` and `e`
* `4-print_hexa.py` – Print decimal and hexadecimal numbers
* `5-print_comb2.py` – Print numbers from `00` to `99`
* `6-print_comb3.py` – Print unique two-digit combinations

---

### 🧠 Functions

* `7-islower.py` – Check lowercase character
* `8-uppercase.py` – Print string in uppercase
* `9-print_last_digit.py` – Print last digit of a number
* `10-add.py` – Add two integers
* `11-pow.py` – Calculate power
* `12-fizzbuzz.py` – Print FizzBuzz sequence

---

### ⭐ Advanced

* `100-print_tebahpla.py` – Print alphabet in reverse with alternating case
* `101-remove_char_at.py` – Remove a character from a string by position
