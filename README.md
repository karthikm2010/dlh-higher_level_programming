## 📘 Objective

To build a strong foundation in Python by learning:

- Output formatting and printing  
- String manipulation  
- Control flow (conditions & loops)  
- Functions and algorithms  
- Modules and imports  
- Command-line arguments  
- Data structures  
- Exception handling
